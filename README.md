#Divisor de Gastos Web
Una aplicación web sencilla y eficiente para dividir gastos entre varias personas. Permite agregar personas, registrar gastos y calcular automáticamente cuánto debe pagar o recibir cada persona.

#Características
Agregar personas: Crea un listado de participantes en el gasto.
Registrar gastos: Añade gastos indicando monto, descripción y quién lo pagó.
División automática: Calcula cuánto le corresponde pagar o recibir a cada persona.
Resumen claro: Visualiza un resumen de los gastos y deudas de cada participante.
Interfaz amigable: Fácil de usar y de entender.

#Tecnologías
Frontend: Vite React JS
Almacenamiento: LocalStorage , JSON
